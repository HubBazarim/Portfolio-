Programa para Junção de Arquivos PDF

Este projeto consiste em um programa em Python que utiliza a biblioteca PyPDF2 para combinar múltiplos arquivos PDF em um único documento.

O código organiza os arquivos PDF em ordem alfabética e os junta de forma eficiente, gerando um arquivo final denominado "PDF Final.pdf".

Ideal para quem precisa consolidar documentos de forma prática e automatizada.
