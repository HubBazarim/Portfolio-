Comunicação em Tempo Real com Python e Flet

Este projeto é um aplicativo de chat em tempo real, desenvolvido com a biblioteca Flet utilizando a linguagem Python. Ele foi projetado para proporcionar uma comunicação interativa entre os usuários de forma prática e eficiente.
- Para que serve? Permitir que múltiplos usuários interajam em uma sala de chat em tempo real, com funcionalidades simples e feedback visual sobre mensagens e novas entradas.
- Como usar? 1. Certifique-se de ter o Python instalado na versão 3.9 ou superior.
2. Instale a biblioteca Flet com o comando: pip install flet.
3. Execute o arquivo principal do projeto com: python main.py.
4. Acesse o aplicativo no navegador através do endereço http://localhost:8000.
5. Clique no botão "Iniciar Chat", insira seu nome no pop-up e comece a interagir no chat enviando mensagens.
- O que foi usado?

  • Linguagem: Python
  • Biblioteca: Flet (para criação de interfaces interativas)
  • Sistema PubSub: Sincronização de mensagens em tempo real
  • Interface Responsiva: AlertDialog, Column, Rows, e ElevatedButton
