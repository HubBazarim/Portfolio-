Projeto de Junção de Arquivos PDF

Automatizando o Processo de Combinação de Múltiplos PDFs em um Único Documento

Vídeo de Funcionamento:

resumo do Projeto:

Este projeto consiste em um programa desenvolvido em Python para juntar múltiplos arquivos PDF em um único documento.
Utilizando a biblioteca PyPDF2, o programa realiza a leitura de todos os arquivos PDF presentes em uma pasta, ordena-os alfabeticamente e os combina em sequência.
O código utiliza a função PdfMerger para anexar os arquivos PDF, gerando um arquivo final denominado "PDF Final.pdf".
Essa automação pode ser útil para quem precisa consolidar vários documentos em um único arquivo, proporcionando uma maneira prática e eficiente de organizar PDFs.
O processo visa facilitar o gerenciamento de documentos, principalmente quando se lida com grandes volumes de arquivos PDF, oferecendo uma solução simples e eficaz para compactar informações dispersas em um único arquivo organizado.
